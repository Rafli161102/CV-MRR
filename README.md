🎨 Tentang Proyek Ini
Ini adalah website portofolio yang dibangun menggunakan Next.js dan Tailwind CSS. Website ini dirancang agar sangat mudah diedit oleh desainer (tanpa perlu paham bahasa pemograman yang rumit). Semua teks, gambar, dan pengalaman kerja dipisahkan ke dalam satu file data sederhana.
🛠️ Cara Edit Konten (Tanpa Koding!)
Jika kamu ingin memperbarui pengalaman kerja, mengganti gambar portofolio, atau mengubah nomor WhatsApp, kamu tidak perlu membongkar desain website.
Buka folder src/data/
Klik file cv.js
Klik tombol Edit (ikon pensil) di GitHub.
Ubah teks yang ada di dalam tanda kutip " ".
Scroll ke bawah dan klik tombol hijau Commit changes.
Website di Vercel akan otomatis terupdate dalam waktu 1 menit! 🌊
🚀 Panduan Deploy Tanpa Terminal (Khusus Pemula)
Jika kamu belum membuat websitenya dan tidak paham menggunakan Terminal/CMD, ikuti langkah super mudah ini:
Langkah 1: Buat Pondasi Instan dari Vercel
Buka Vercel.com dan Login dengan GitHub.
Di Dashboard, klik tombol Add New... lalu pilih Project.
Di menu sebelah kiri, cari tulisan Clone Template dan klik logo Next.js.
Beri nama repositori kamu (misal: portofolio-rafli) dan klik Create.
Vercel akan secara otomatis membuatkan repository Next.js di GitHub kamu dan langsung online!
Langkah 2: Timpa dengan Desain Portofoliomu
Buka akun GitHub kamu dan masuk ke repository portofolio-rafli yang baru saja dibuat oleh Vercel.
Masuk ke folder src -> app.
Buka file page.js atau page.jsx, klik Edit (ikon pensil).
HAPUS SEMUA isinya, dan Paste (tempel) kode desain portofolio yang telah diberikan. (Klik Commit changes).
Selesai! Buka link Vercel kamu dan lihat hasilnya!
