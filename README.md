​<h2>🎨 Tentang Proyek Ini</h2>
<p>Ini adalah website portofolio yang dibangun menggunakan Next.js dan Tailwind CSS. Website ini dirancang agar <b>sangat mudah diedit oleh desainer (tanpa perlu paham bahasa pemograman yang rumit)</b>. Semua teks, gambar, dan pengalaman kerja dipisahkan ke dalam satu file data sederhana.</p>
​<hr>
​<h2>📂 Struktur Website</h2>
<p>Berikut adalah susunan folder proyek ini agar kamu tahu di mana letak file-file penting:</p>
<ul>
<li>📁 <b>public/</b> <i>(Taruh SEMUA gambar & foto karyamu di sini)</i></li>
<li>📁 <b>src/</b>
<ul>
<li>📁 <b>app/</b>
<ul>
<li>📄 <code>layout.jsx</code> <i>(Kerangka utama website)</i></li>
<li>📄 <code>page.jsx</code> <i>(Halaman utama desain UI portofolio)</i></li>
<li>📄 <code>globals.css</code> <i>(File CSS & Tailwind)</i></li>
</ul>
</li>
<li>📁 <b>data/</b>
<ul>
<li>📝 <code>cv.js</code> <b>(📍 TEMPAT KAMU MENGEDIT SEMUA TEKS & LINK)</b></li>
</ul>
</li>
</ul>
</li>
</ul>
​<hr>
​<h2>🛠️ Cara Edit Konten (Tanpa Koding!)</h2>
<p>Jika kamu ingin memperbarui pengalaman kerja, mengganti gambar portofolio, atau mengubah nomor WhatsApp, kamu <b>tidak perlu membongkar desain website</b>.</p>
<ol>
<li>Buka folder <code>src/data/</code></li>
<li>Klik file <code>cv.js</code></li>
<li>Klik tombol <b>Edit (ikon pensil)</b> di GitHub.</li>
<li>Ubah teks yang ada di dalam tanda kutip <code>" "</code>.</li>
<li>Scroll ke bawah dan klik tombol hijau <b>Commit changes</b>.</li>
<li>Website di Vercel akan otomatis terupdate dalam waktu 1 menit! 🌊</li>
</ol>
​<hr>
​<h2>🚀 Panduan Deploy Cepat</h2>
<p>Jika kamu belum membuat websitenya, ikuti langkah super mudah ini:</p>
<ol>
<li>Buka <b>Vercel.com</b> dan Login dengan GitHub.</li>
<li>Di Dashboard, klik tombol <b>Add New...</b> lalu pilih <b>Project</b>.</li>
<li>Di menu sebelah kiri, cari tulisan <b>Clone Template</b> dan klik logo <b>Next.js</b>.</li>
<li>Beri nama repositori kamu (misal: <code>portofolio-rafli</code>) lalu klik <b>Create</b>.</li>
<li>Buka akun GitHub kamu, masuk ke repo yang baru dibuat tersebut.</li>
<li>Masuk ke folder <code>src</code> -> <code>app</code> -> buka <code>page.js</code>.</li>
<li><b>HAPUS SEMUA</b> isinya, dan <i>Paste</i> (tempel) kode desain UI yang sudah saya berikan sebelumnya.</li>
<li>Selesai! Buka link Vercel kamu dan lihat hasilnya!</li>
</ol>
